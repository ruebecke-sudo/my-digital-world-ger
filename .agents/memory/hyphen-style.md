---
name: Bindestrich-Stil (Rüdiger Becker / My Digital World)
description: Benutzer möchte keine Bindestriche in zusammengesetzten Wörtern und keine Gedankenstriche in Texten auf der Website.
---

## Regel
In allen sichtbaren Texten (beschreibung, description, badge, label, statLabel, Fließtext) auf der Website:

- **Gedankenstriche** (` – `, ` — `) → durch Komma ersetzen: `, `
- **Bindestriche in Compound-Wörtern** → durch Leerzeichen ersetzen

**Beispiele:**
- `KI-gestützt` → `KI gestützt`
- `All-in-One` → `All in One`
- `No-Code` → `No Code`
- `Full-Stack` → `Full Stack`
- `Social-Media-Plattform` → `Social Media Plattform`
- `Design-Inspiration` → `Design Inspiration`
- `Open-Source` → `Open Source`
- `UI-Kits` → `UI Kits`
- `App-Screens` → `App Screens`
- Gedankenstrich: `„... einsetzt – von KI..."` → `„... einsetzt, von KI..."`

**Ausnahme:** `E-Mail` bleibt mit Bindestrich (offizielle deutsche Schreibweise laut Duden).

**Why:** Benutzer findet viele Bindestriche unschön und unleserlich. Gilt für alle Seiten der Website.

**How to apply:** Immer wenn neue Texte geschrieben oder bestehende Texte bearbeitet werden — von Anfang an keine Bindestriche in Compound-Wörtern verwenden, außer bei E-Mail.
